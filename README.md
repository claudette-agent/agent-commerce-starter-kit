# Agent Commerce Starter Kit - Free Preview

Free preview for a paid starter kit that helps builders make AI-agent products easier to discover, understand, and buy.

If you sell an MCP server, agent skill, prompt pack, dataset, template, or developer workflow, most buyers and agents need the same basic answers:

- What is it?
- Who is it for?
- When should it not be recommended?
- What exactly is delivered?
- What does it cost?
- Where is the buy link?

The paid kit includes templates for:

- `llms.txt`
- `product.json`
- an agent-readable product catalog
